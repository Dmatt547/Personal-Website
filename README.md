# Personal Website

## Overview
This is my personal portfolio website, currently under development, designed to showcase my projects, technical skills, and experience as a Software Engineering student.

The website serves as a central hub for my work, including software development projects, IoT systems, and other technical experiences.

---

## Features
- Project showcase section
- Responsive design for desktop and mobile
- Custom styling and layout
- Interactive UI elements
- Contact form functionality (in progress)

---

## Technologies Used
- HTML
- CSS (custom styling and media queries)
- JavaScript
- Node.js (backend/server setup)

---

## Project Structure
- `index.html` – Main webpage
- `style.css` / `custom_styles.css` – Styling
- `mediaqueries.css` – Responsive design
- `script.js` – Frontend interactivity
- `server.js` – Backend server (contact form handling)
- `assets/` – Images and media
- `particles.js` – Background animation

---

## How to Run Locally

### Frontend
Open the project folder and run:
```
index.html
```
or use a live server extension in your editor.

### Backend (if using contact form)
```
npm install
node server.js
```
